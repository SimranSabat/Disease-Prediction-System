🏥 Disease Prediction System

A full-stack web application that predicts diseases based on patient symptoms, built with Django and Machine Learning.

## 🚀 Features
- Symptom-based disease prediction using ML
- User authentication (login, dashboard, reports)
- Admin panel for disease management
- File upload support

## 🛠️ Tech Stack
- **Backend:** Python, Django
- **ML:** scikit-learn
- **Frontend:** HTML, CSS, Bootstrap, jQuery
- **Database:** SQLite

## ⚙️ Setup
\`\`\`bash
cd disease_prediction_system
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
\`\`\`
